## Mod Types

### Categories

1. **Construction/Decoration** - Decorative blocks, furniture, windows, doors, bridges
2. **Quality of Life (QoL)** - Inventory management, improved storage, automation
3. **Gameplay** - New game modes (Skyblock, etc.), advanced farming, music systems
4. **Server/Multiplayer** - Land protection, economies, mini-games
5. **Visuals** - Texture Packs (16x to 512x), Shaders (including Ray Tracing), PBR Packs

### Technical Types

| Type | Description | Requirements |
|------|-------------|--------------|
| **Packs** | Asset/content packs for new blocks, mobs, items | No code - Asset Editor |
| **Plugins** | Java mods using the game API | Java programming |
| **Bootstrap Plugins** | Low-level plugins for bytecode transformation | Advanced Java |

## Key Information

### Server-Side Architecture
- **All mods run server-side**
- No client installation required
- Players automatically receive content when connecting
- Server is not obfuscated (decompilation possible)

### CurseForge Partnership (January 5, 2026)
- Direct integration with Hytale client
- One-click installation
- Proactive moderation and security scanning
- Real-time adoption statistics for creators
