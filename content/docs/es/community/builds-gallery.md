## Creation Tools in Hytale

Hytale offers a comprehensive suite of building tools designed for creators of all skill levels.

### Core Building Tools

| Tool | Description | Best For |
|------|-------------|----------|
| **Creative Mode** | Unlimited access to all blocks and items | Free building without constraints |
| **Scripted Brushes** | Procedural generation brushes | Large-scale terraforming, repetitive patterns |
| **Prefabs** | Reusable structure templates | Modular building, consistency |
| **Selection Tool** | Mass copy, paste, and move | Large area manipulation |
| **World Tools** | Terrain editing capabilities | Landscape modification |
| **Machinima Mode** | Cinematic capture system | Showcasing builds |

### Scripted Brushes

Scripted Brushes are one of Hytale's most powerful features for builders:

- **Procedural Generation** - Create complex patterns automatically
- **Custom Algorithms** - Define your own generation rules
- **Terrain Sculpting** - Shape landscapes naturally
- **Vegetation Placement** - Populate areas with flora

### Prefabs System

The Prefab system allows builders to:

- **Save Structures** - Store creations for later use
- **Share Templates** - Export and share with others
- **Modular Design** - Build with reusable components
- **Quick Placement** - Rapidly place saved structures

### Selection Tool

Perfect for large-scale projects:

- **Area Selection** - Select large regions
- **Copy & Paste** - Duplicate sections
- **Move Structures** - Relocate entire builds
- **Rotate & Mirror** - Transform selections

## Tips for Impressive Builds

### Planning Your Build

1. **Concept First**
   - Sketch your idea before building
   - Define the style and theme
   - Plan the scale and proportions

2. **Reference Materials**
   - Gather inspiration images
   - Study real architecture
   - Look at other builders' work

3. **Start with Structure**
   - Build the skeleton first
   - Establish key dimensions
   - Work from large to small details

### Building Techniques

#### Depth and Dimension

- **Layering** - Add multiple levels to walls
- **Overhangs** - Create shadows with protruding elements
- **Recesses** - Add depth with inset areas
- **Varied Heights** - Avoid flat rooflines

#### Lighting Effects

- **Natural Light** - Consider sun positioning
- **Light Sources** - Place torches and lanterns strategically
- **Shadow Play** - Use lighting to create atmosphere
- **Accent Lighting** - Highlight key features

#### Detailing

- **Texture Variation** - Mix block types
- **Small Additions** - Buttons, trapdoors, chains
- **Functional Elements** - Add believable details
- **Color Gradients** - Blend colors naturally

#### Landscaping

- **Terrain Integration** - Blend builds with environment
- **Vegetation** - Add plants and trees
- **Paths and Roads** - Connect areas naturally
- **Water Features** - Incorporate rivers, ponds

## How to Submit Your Build

### Submission Criteria

To be featured in this gallery, your build should meet these standards:

| Criteria | Requirement |
|----------|-------------|
| **Originality** | Your own creation, not copied |
| **Quality** | Well-crafted with attention to detail |
| **Completion** | Finished or substantially complete |
| **Screenshots** | High-quality images required |
| **Description** | Include build information |

### Required Information

When submitting, please provide:

1. **Build Details**
   - Name of your creation
   - Category (from list above)
   - Brief description (50-200 words)
   - Build time estimate
   - Tools used

2. **Creator Information**
   - Your builder name/alias
   - Social media links (optional)
   - Previous notable builds (optional)

3. **Screenshots**
   - Minimum 3 high-quality images
   - Various angles and distances
   - Day and night shots (if applicable)
   - Detail shots of key features

### How to Take Great Screenshots

:::tip Screenshot Guidelines
- Use Hytale's Screenshot Mode for best quality
- Disable UI elements
- Choose optimal lighting conditions
- Experiment with camera angles
- Consider using shaders if available
:::

**Recommended Settings:**
- High resolution (1920x1080 minimum)
- Optimal render distance
- Good weather conditions
- Strategic time of day

### Submission Process

1. **Prepare Your Submission**
   - Gather all required information
   - Take quality screenshots
   - Write your description

2. **Submit via Wiki Contribution**
   - Follow our [Contributing Guide](/docs/community/contributing)
   - Create a pull request with your build

3. **Community Review**
   - Builds are reviewed for quality
   - Feedback may be provided
   - Approved builds are added to the gallery
