# Animations

Animate your models using Blockbench.

## Format

Animations use `.blockyanim` files.

## Creating Animations

1. Set up bone hierarchy
2. Create animation in Blockbench
3. Set keyframes
4. Use graph editor for curves
5. Export as `.blockyanim`

See [Animation Guide](/docs/tools/blockbench/animation).
