# Block Behaviors

Add interactive behaviors to your blocks.

## Available Behaviors

- **Gravity** - Block falls like sand
- **Liquid** - Flows like water
- **Interactive** - Responds to player actions
- **Emissive** - Produces light

## Example

```json
{
  "behaviors": [
    {
      "type": "emissive",
      "light_level": 15
    }
  ]
}
```

*Documentation will be expanded as more behaviors are documented.*
