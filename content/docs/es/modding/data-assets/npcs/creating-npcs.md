# Creating NPCs

Create custom NPCs for your world.

## Basic NPC Definition

```json
{
  "id": "my_mod:custom_creature",
  "displayName": "Custom Creature",
  "health": 20,
  "model": "models/custom_creature.blockymodel"
}
```
