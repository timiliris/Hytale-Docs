# Item Types

Overview of available item types.

| Type | Description |
|------|-------------|
| `weapon` | Deals damage |
| `tool` | Mining/harvesting |
| `armor` | Protection |
| `consumable` | Food/potions |
| `material` | Crafting ingredients |
