# Creating Items

Learn how to create custom items.

## Basic Item Definition

```json
{
  "id": "my_mod:ruby_sword",
  "displayName": "Ruby Sword",
  "type": "weapon",
  "properties": {
    "damage": 10,
    "durability": 500
  }
}
```
