# Blockbench Installation

Blockbench is the official tool for creating 3D models and animations for Hytale. It replaces the internal "Hytale Model Maker" tool to better support established creative workflows in the community.

> "We officially support Blockbench for creating Hytale models, textures, and animations."
> -- [Hytale Modding Strategy](https://hytale.com/news/2025/11/hytale-modding-strategy-and-status)

## Download Blockbench

Get Blockbench from the official website: [blockbench.net](https://www.blockbench.net/)

**Available platforms:**
- Windows
- macOS
- Linux
- Web version (limited features)

## Install the Hytale Plugin

The official Hytale plugin adds support for Hytale-specific formats and features.

### From Plugin Manager

1. Open Blockbench
2. Go to **File > Plugins**
3. Click on the **Available** tab
4. Search for "Hytale"
5. Click **Install**

### From GitHub

Alternatively, download directly from the official repository:

**GitHub Repository:** [https://github.com/JannisX11/hytale-blockbench-plugin](https://github.com/JannisX11/hytale-blockbench-plugin)

1. Download the plugin file
2. In Blockbench, go to **File > Plugins**
3. Click the **Load Plugin from File** button
4. Select the downloaded plugin file

## Verify Installation

To confirm the plugin is installed correctly:

1. Open Blockbench
2. Go to **File > New**
3. Look for **Hytale Model** in the project types
4. Create a new Hytale Model project

If you see the Hytale Model option, the plugin is installed successfully.

## Supported Formats

The Hytale plugin enables:

| Format | Purpose |
|--------|---------|
| `.blockymodel` | 3D model files |
| `.blockyanim` | Animation files |
