# Machinima Tools

Hytale includes built-in Machinima tools for recording content and creating cinematics. These are the same tools used to create Hytale's official trailers.

> "The Machinima Tools have many more features not displayed in quick demos, and they are scheduled for a massive upgrade in functionality and user experience in the coming months after launch."
> -- [Hytale Creative Mode](https://hytale.com/news/2025/11/hytale-creative-mode)

## Features

### Camera Actor

Create camera entities for cinematic shots.

**Capabilities:**
- Keyframe-based animation
- Define camera trajectories
- Set custom speeds and easing
- Create complex camera movements

### Recording System

Capture gameplay and scenes.

**Features:**
- Record gameplay footage
- Capture cinematic sequences
- Integration with camera actors

### Playback Controls

Review and refine your creations.

**Features:**
- Play back recorded sequences
- Preview camera movements
- Adjust timing and pacing

### Export Options

Output your content for sharing.

**Features:**
- Export video content
- Share cinematics
- Create trailers and promotional content

## Creating Cinematics

### Basic Workflow

1. **Plan your shots** - Storyboard your cinematic
2. **Set up the scene** - Use Creative Mode to build your environment
3. **Place camera actors** - Position cameras for each shot
4. **Define keyframes** - Animate camera movement and timing
5. **Record** - Capture the sequence
6. **Review and refine** - Adjust as needed
7. **Export** - Output the final video

### Camera Animation

Cameras use keyframe animation:

- **Position keyframes** - Where the camera is located
- **Rotation keyframes** - Where the camera is pointing
- **Speed curves** - Control acceleration and deceleration
- **Custom behaviors** - Define special camera effects

## Use Cases

- **Trailers** - Create promotional videos for your content
- **Storytelling** - Narrative cinematics for adventure maps
- **Tutorials** - Educational content showing game features
- **Machinima films** - Full video productions within the game
- **Screenshots** - Position cameras for perfect screenshots

## Integration with Creative Mode

Machinima tools work seamlessly with Creative Mode:

- Use the NPC browser to place actors
- Build sets with unlimited blocks
- Use prefabs for quick scene setup
- Fly around to scout locations

## Tips for Creators

1. **Plan ahead** - Storyboard before you start
2. **Use smooth curves** - Avoid jerky camera movements
3. **Test frequently** - Preview your work often
4. **Layer shots** - Combine multiple camera angles
5. **Consider lighting** - Time of day affects mood
