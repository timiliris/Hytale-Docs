# Creative Mode Tools

Creative Mode is available at launch and provides access to all of Hytale's creation tools for building and editing worlds.

> "By swapping to Creative Mode in your world, you can fly around and explore, create, edit, make things explode, and more."
> -- Ktar, Creative Mode Engineer

## Features Overview

Creative Mode gives players:
- **Unlimited access to all blocks** - Build without resource gathering
- **Advanced building tools** - Professional-grade creation capabilities
- **Character model changer** - Access to 100+ different NPC models for content creation
- **All creation tools available** - Full access to the Hytale toolset

## Available Tools

### Selection Tool

Advanced selection tool for manipulating regions of the world.

**Features:**
- Select 3D regions
- Copy and paste selections
- Move and rotate selections
- Works with other editing tools

### Paintbrush

Paint shapes and materials directly into the world.

**Features:**
- Shape options: pyramids, cones, squares, and more
- Customizable materials
- Adjustable density
- Mask support for precise painting

### Prefab System

Work with pre-built reusable structures.

**Features:**
- Load prefabs from a library of 100+ structures
- Edit prefabs in a separate editing environment
- Save custom structures as prefabs
- Share prefabs between worlds

### NPC Browser

Browse and spawn NPCs for testing and content creation.

**Features:**
- Access to all game NPCs
- Model changer with 100+ character models
- Useful for machinima and screenshots

### Flight Controls

Customizable flight for navigating and building.

**Options:**
- Adjustable flight speed
- Custom controls
- Flight mode selection (hover/directional)
- Inertia settings

## Scripted Brushes

Procedural brushes for creating complex terrain and structures automatically.

**Available Brush Types:**
- **Mountain brushes** - Generate realistic mountain formations
- **Path brushes** - Create winding paths and roads
- **Ruins brushes** - Place procedurally generated ruins
- **Custom brushes** - Create your own using the Node Editor

Scripted brushes use the Visual Scripting system to define procedural generation rules.

## World Tools Integration

Creative Mode integrates with Hytale's World Tools for comprehensive world editing:

- Terrain modification
- Biome editing
- Structure placement
- World generation previewing

## Visual Scripting Access

From Creative Mode, you can access the Visual Scripting system:

- Create custom behaviors
- Design game logic without code
- Build custom brushes and tools
- Inspired by Unreal Engine's Blueprints

## Tips for Creators

1. **Use prefabs** - Save frequently-used structures as prefabs for quick placement
2. **Scripted brushes for terrain** - Use procedural brushes for natural-looking landscapes
3. **Test with NPC browser** - Spawn NPCs to test scale and proportions
4. **Customize flight** - Adjust flight settings for your workflow
