# Puntos Finales de la API

Puntos finales oficiales de la API de Hytale.

## URL Base

```
https://hytale.com/api
```

## Puntos Finales

### Publicaciones del Blog

```
GET /blog/post/published
```

Devuelve las publicaciones del blog que han sido publicadas.

### Publicación por Slug

```
GET /blog/post/slug/{slug}
```

Devuelve una publicación específica.
