# Liste des Serveurs Communautaires

Guide complet des serveurs communautaires Hytale disponibles depuis le lancement de l'Early Access (janvier 2026).

## Introduction

Les serveurs communautaires sont disponibles des le lancement, offrant une variete d'experiences de jeu au-dela des modes officiels.

### Caracteristiques Cles

- **Mods cote serveur** : Tous les mods s'installent automatiquement quand vous rejoignez un serveur - aucun telechargement manuel requis
- **Acces facile** : Rejoignez les serveurs directement via le launcher Hytale
- **Variete** : De la survie au roleplay, trouvez le style de jeu qui vous convient

### Comment Rejoindre un Serveur

1. Ouvrez le launcher Hytale
2. Naviguez vers l'onglet "Serveurs"
3. Parcourez ou recherchez un serveur
4. Cliquez sur "Rejoindre" - les mods se telechargeront automatiquement
5. Commencez a jouer !

## Comment Choisir son Serveur

### 1. Ping et Latence

Choisissez un serveur geographiquement proche de vous pour une meilleure experience :
- **< 50ms** : Excellente connexion
- **50-100ms** : Bonne connexion
- **100-200ms** : Jouable
- **> 200ms** : Risque de lag

### 2. Type de Gameplay

Reflechissez a ce que vous voulez faire :
- **Fans de PvP** : Recherchez des serveurs survie avec PvP active
- **Constructeurs** : Serveurs creatifs ou survie avec protection de terrain
- **Joueurs sociaux** : Serveurs roleplay ou axes communaute
- **Joueurs competitifs** : Serveurs mini-jeux avec classements

### 3. Communaute et Moderation

- Verifiez si le serveur a des moderateurs actifs
- Lisez les regles du serveur avant de rejoindre
- Cherchez les communautes Discord pour evaluer l'activite
- Consultez les avis des joueurs sur les sites de listing

### 4. Mods Installes

- Certains serveurs sont en vanilla Hytale
- D'autres ont des packs de mods extensifs
- Les mods se telechargent automatiquement, mais les gros packs prennent plus de temps
- Verifiez si les mods correspondent a vos preferences

## Creer son Propre Serveur

Vous voulez heberger votre propre serveur Hytale ?

### Documentation

Consultez notre guide complet d'hebergement : [Documentation Serveurs](/docs/technical/servers)

### Options d'Hebergement

| Option | Avantages | Inconvenients |
|--------|-----------|---------------|
| **Auto-heberge** | Controle total, gratuit (hors materiel) | Necessite des connaissances techniques |
| **VPS/Cloud** | Flexible, evolutif | Cout mensuel, configuration requise |
| **Hebergeur de jeux** | Installation facile, gere | Cout mensuel, moins de controle |
| **Serveur dedie** | Performance maximale | Cout plus eleve |

### Configuration de Base

1. Telechargez les fichiers du serveur Hytale
2. Configurez `server.properties`
3. Configurez le port forwarding (si auto-heberge)
4. Installez les mods souhaites
5. Partagez l'IP de votre serveur avec les joueurs

## Regles Communautaires

### Pour les Joueurs

- **Respectez les regles du serveur** : Chaque serveur a ses propres directives
- **Signalez les comportements toxiques** : Aidez a maintenir une communaute saine
- **Pas de triche ni de hack** : Jouez fair-play, respectez les autres joueurs
- **Soyez respectueux** : Traitez les autres comme vous aimeriez etre traite

### Pour les Proprietaires de Serveurs

- Affichez et appliquez clairement les regles
- Ayez une moderation active
- Fournissez un moyen de signaler les problemes
- Gardez votre serveur a jour et securise

## FAQ - Serveurs

### Comment voir le ping d'un serveur ?

Le navigateur de serveurs dans le launcher Hytale affiche les informations de ping pour chaque serveur. Vous pouvez egalement trier les serveurs par ping pour trouver les plus proches.

### Les mods sont-ils obligatoires ?

Non ! Quand vous rejoignez un serveur modde, les mods requis sont automatiquement telecharges et installes. Quand vous quittez, votre client revient a l'etat vanilla. Aucune installation manuelle necessaire.

### Puis-je jouer avec des amis sur differents serveurs ?

La progression est generalement specifique a chaque serveur. Votre personnage sur le Serveur A est separe du Serveur B. Cependant, certains serveurs pourraient offrir des fonctionnalites cross-serveur a l'avenir.

### Comment signaler un serveur ?

Si un serveur viole les conditions d'utilisation de Hytale, vous pouvez le signaler via :
- Le systeme de signalement en jeu
- Les canaux de support officiels de Hytale
- Le site de listing ou vous l'avez trouve

### Puis-je avoir plusieurs personnages ?

Cela depend des regles du serveur. Certains autorisent plusieurs comptes, tandis que d'autres exigent un personnage par joueur.

### Et si un serveur a trop de mods ?

Les gros packs de mods prennent plus de temps a telecharger lors de la premiere connexion. Apres le telechargement initial, les connexions suivantes sont plus rapides. Assurez-vous d'avoir assez d'espace disque.
