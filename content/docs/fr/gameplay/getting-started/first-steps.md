# Premiers pas sur Hytale

Bienvenue sur Orbis ! Ce guide vous accompagnera dans vos premières heures de jeu dans l'Early Access de Hytale.

## Lancement du jeu

### Configuration requise

| Composant | Minimum | Recommandé |
|-----------|---------|------------|
| OS | Windows 10 64-bit (version 1809) | Windows 11 64-bit |
| Processeur | Intel i5-7500 / AMD Ryzen 3 1200 | Intel i5-10400 ou supérieur |
| RAM | 8 Go (12 Go avec carte graphique intégrée) | 16 Go |
| GPU | GTX série 900 / RX série 400 / Intel Arc série A | RTX série 20 / RX série 5000 |
| Stockage | 8 Go (20 Go avec données de monde) | 20+ Go SSD |

**Note de performance**

Hytale est un jeu sandbox voxel qui dépend fortement des performances du CPU et de la RAM. Une fois que votre GPU atteint les spécifications minimales, améliorer votre CPU et avoir suffisamment de mémoire système vous aidera davantage qu'une carte graphique plus puissante.

### Premier lancement

1. Téléchargez Hytale depuis [hytale.com](https://hytale.com)
2. Installez le jeu
3. Connectez-vous avec votre compte Hytale
4. Choisissez votre mode de jeu (Mode Exploration recommandé pour les débutants)

## Création de personnage

Avant de commencer, personnalisez votre avatar :

- **Apparence** - Visage, coiffure, couleur de peau
- **Vêtements** - Tenue de départ
- **Nom** - Votre identité sur Orbis

## Démarrer votre aventure

Vous apparaissez dans un **bâtiment de départ** qui offre un abri initial.

### Premiers objectifs : Récolter les ressources de base

Sortez et récoltez ces matériaux essentiels :

1. **Fibres végétales** - Cassez les buissons
2. **Bâtons** - Ramassez-les au sol
3. **Gravats de pierre** - Ramassez les rochers au sol
4. **Nourriture** - Baies sauvages, pommes et autres aliments

### Fabriquer vos premiers outils

1. Appuyez sur **Tab** pour ouvrir votre inventaire
2. Cliquez sur l'outil que vous voulez fabriquer
3. Consultez les ingrédients requis
4. Cliquez sur **Fabriquer** quand vous avez les matériaux

Premiers outils essentiels :
- **Hachette** - Pour couper les arbres (tous les blocs d'arbre tombent quand vous coupez le bloc du bas !)
- **Pioche** - Pour miner la pierre et le minerai dans les grottes
- **Arme basique** - Pour vous défendre en combat

### Construire votre premier établi

1. Utilisez votre pioche pour obtenir de la **pierre** dans les grottes
2. Utilisez votre hachette pour obtenir du **bois** (troncs d'arbre)
3. Ouvrez l'inventaire (Tab), trouvez l'**Établi du constructeur**
4. Fabriquez-le avec 6 troncs d'arbre et 3 pierres
5. Placez-le dans votre bâtiment de spawn pour le protéger

**Importance de l'abri**

Les mobs hostiles apparaissent la nuit ! Utilisez votre bâtiment de spawn comme première base provisoire.

## Mécaniques de jeu clés

### Le système de Mémoires

Hytale propose un **système de Mémoires** unique où vous documentez vos rencontres avec les créatures pour débloquer des récompenses :
- L'exploration devient aussi précieuse que la collecte de ressources
- Enregistrez les créatures que vous rencontrez pour des avantages permanents
- Accédez aux Mémoires via le Temple Oublié

**Activez tôt !**

Le système de Mémoires n'est pas rétroactif. Vous devez l'activer tôt, sinon vous manquerez la documentation des créatures rares rencontrées avant d'avoir débloqué le système.

### L'économie d'Essence

Collectez deux types d'Essence :
- **Essence de Vie** - Des plantes et animaux pendant le jour
- **Essence du Vide** - Des créatures démoniaques la nuit

Ces ressources alimentent les stations de fabrication avancées et les échanges avec les PNJ au hub du Temple Oublié.

### Mécaniques de mort

Hytale a des mécaniques de mort indulgentes :
- Vous perdez certains objets à la mort
- Les objets restants subissent 10% de perte de durabilité
- Votre emplacement de mort apparaît comme un **marqueur de tombe** sur votre carte
- La récupération des objets est simple

## Objectif du début de jeu : Portail du Temple Oublié

Le **Portail du Temple Oublié** est votre objectif le plus crucial en début de jeu :

1. Ouvrez votre carte (touche **M**)
2. Cherchez l'**icône spirale** marquant le Portail du Temple Oublié
3. Préparez-vous avant d'y aller :
   - Armes en cuivre
   - Un bouclier
   - Nourriture cuite
   - Armure basique si possible
4. Voyagez jusqu'à l'emplacement et descendez dans le temple
5. Vainquez le gardien golem
6. Débloquez l'accès au monde hub, au commerce et au système de Mémoires

## Transport : Chevaux

Contrairement à d'autres jeux, monter à cheval dans Hytale est simple :
- Trouvez un cheval adulte
- Appuyez sur **F** pour monter
- Sprintez pour voyager plus vite
- Les chevaux restent généralement où vous les laissez
- Construisez une écurie provisoire (enclos en terre/bois) pour les garder en sécurité

## Conseils pour bien démarrer

### Exploration

- Repérez les **points d'intérêt** sur votre carte
- Les colonnes de fumée indiquent des camps
- Les structures cachent souvent des trésors et du butin

### Combat

- Remplissez votre **barre de compétence d'arme** grâce aux coups réussis
- Appuyez sur **Q** quand elle est pleine pour une attaque spéciale dévastatrice
- L'endurance se vide pendant le blocage mais pas pendant l'attaque
- Apprenez les patterns d'attaque des ennemis avant de les affronter

### Ressources

- Récoltez tout ce que vous trouvez au début
- Les coffres du monde contiennent des objets utiles
- Parlez aux PNJ pour des quêtes et informations

## Prochaines étapes

Une fois les bases maîtrisées :

1. Atteignez le **Portail du Temple Oublié** et vainquez le gardien
2. Améliorez votre **Établi** pour débloquer plus de recettes
3. Fabriquez l'**Établi de l'armurier** pour un meilleur équipement
4. Apprenez la **cuisine** au Fourneau du chef pour des bonus
5. Explorez les **donjons** pour un meilleur butin
6. Découvrez de nouvelles **régions** d'Orbis

**Sauvegarde automatique**

Hytale sauvegarde automatiquement votre progression. En mode solo, vous pouvez aussi créer des sauvegardes manuelles.
